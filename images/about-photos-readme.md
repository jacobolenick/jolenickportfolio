# About Page Photos

Please add your about page photos to the `/images/` directory with these names:

- `about-photo-1.jpg` - Main photo (front photo in the stack)
- `about-photo-2.jpg` - Middle photo in the stack  
- `about-photo-3.jpg` - Back photo in the stack

## Recommended Specs:
- Size: 800x800px (square)
- Format: JPG or PNG
- Max file size: 500KB each

The photos will appear in a stacked, tilted card layout on the about page.
