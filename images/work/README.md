# Work Project Images

Please save your project cover images to this directory with the following names:

## Required Images:

1. **criminal-minds.jpg** - Criminal Minds Paramount+ streaming project screenshot
2. **publix.jpg** - Publix e-commerce website screenshot  
3. **ticketing.jpg** - Event ticketing/seating system screenshot
4. **design-system.jpg** - Design system and UI components screenshot
5. **banking.jpg** - Financial services/banking application screenshot

## Image Specifications:

- **Format**: JPG or PNG (update the file extension in `index.html` if using PNG)
- **Recommended size**: 1200px x 800px (3:2 aspect ratio)
- **Max file size**: Keep under 500KB for best web performance

## To Update:

1. Save each screenshot from your conversation with the exact filenames listed above
2. Place them in this `/images/work/` directory
3. If you use PNG instead of JPG, update the file extensions in `index.html`:
   - Find: `images/work/filename.jpg`
   - Replace with: `images/work/filename.png`

The images will automatically display in the Work section of your portfolio!
